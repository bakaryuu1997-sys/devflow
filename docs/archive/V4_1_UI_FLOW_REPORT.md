# v4.1 UI Flow Report

## Goal

Make the interface easy to use, easy to understand, and clearly ordered.

## Added

- 6-step wizard.
- Sidebar with recommended order.
- `Next recommended action` button.
- Step-specific panels.
- Clear metric cards.
- Predictable result panels.
- Stronger color meaning.
- Clear separation between artifact scanning, traceability, impact and release gate.

## User flow

```text
1. Reset demo
2. Upload artifacts
3. Run traceability
4. Record change and impact
5. Run advanced release gate
6. Export evidence
```
